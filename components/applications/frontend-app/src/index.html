Infra as Code App Demo

Esta página tem a intenção de simular uma aplicação frontend servida através de um bucket no Google Cloud Storage.

Tecnologias usadas neste projeto

O conteúdo abaixo foi carregado através da aplicação backend, que responde no caminho "/api/*" deste domínio. Ao executar esta API em um ambiente com múltiplas réplicas, o hostname é alterado ao recarregar a página. No Kubernetes por exemplo, o hostname mostrado corresponde ao nome do pod que respondeu a solicitação.
